core.Modes    (function)  
    Allows you to set multiple modes with one statement.

File  
    pops/pops.core.js

Syntax  
Modes(nodeVals);
Modes({
    node1: val1,
    node2: val2,
    node3: val3,
});
  • modeVals : An object containing the vaues for nodes, where each members name is a mode name.
  • mode1, node2, node3 : The node's name.
  • val1, val2, val3 : The node's value.
