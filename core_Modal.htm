core.Modal    (function)  
    Modal returns a result from 'choices' based on the 'mode' argument. This is the same as Choose, except it gets its 'choice' value form a Mode specified by the 'mode' argument instead of using a 'choice' argument.

File  
    pops/pops.core.js

Syntax  
var foo = Modal(mode, choices);
  • mode : A string value representing a Mode.
  • choices : An object containing the choice values. Each member is a choice, it's name is the choice's name. You can use multiple choice names for a single choice using the 'or' operator (_?_).

Choices  
  • $else: The value of this member will be returned if no choices can be matched.  
    $else: val,
    • val : Any value.
  • $run: If set to true, and the resulting value is a function, Choose will run the function and return the result rather than returning the function itself.  
    $run: true,
Naming syntax  
    Naming a choice is like naming a variable, the same rules apply.

Name     Result    
myChoicegood
my Choiceerror
myChoice2good
2myChoiceerror
my_Choicegood
$myChoicegood
my@Choiceerror
?myChoiceerror

_or_  
    Use  _or_  to set a single value to multiple choices.
Choice1_or_Choice2_or_MyChoice: val,
  • Choice1, Choice2, MyChoice : These are choice names.
  • val : Any value.

Examples  
Modal  
require('pops/pops.loader').Load();

cout('$os = ' + Mode('$os').Get());

var result = Modal('$os', {
    win32: '- Running on Windows 32',
    linux_or_linux: '- Running on Linux',
    darwin: '- Running on Darwin',
    $else: '- Running on UNKNOWN'
});

cout(result);

Mode('test').Set('hi');

var result2 = Modal('Test', {
    Hello: '- HELLO -',
    Goodbye: '- GOODBYE -',
    Hi: '- HI -',
    Bye: '- BYE -'
});

cout(result2);
