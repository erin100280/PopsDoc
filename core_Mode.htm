core.Mode    (function)  
    returns a reference object to the mode, specified by the 'mode' argument. If the mode does not exsist, a new one will be created.

File  
    pops/pops.core.js

Syntax  
var foo = Mode(mode);
  • mode : A string value representing the name of a mode.

Mode reference  
    An object that references a specific mode.

Members  
  • Get : function - Returns the mode value.  
    var val = modeRef.Get();
    var val = Mode(modeName).Get();
    • modeRef : Any mode reference.
    • modeName : A string value representing the name of a mode. If omitted, "main" will be used.
  • Set : function - Sets the mode value.  
    modeRef.Set(newValue);
    Mode(modeName).Set(newValue);
    • modeRef : Any mode reference.
    • modeName : A string value representing the name of a mode. If omitted, "main" will be used.
    • newValue : A string representing the new value for a mode.

System modes  
    These modes are set by Pops Loader.

  • $os : The operating system Node is running on.
Examples  
Mode & Mode reference  
require('pops/pops.loader').Load();

cout('$os = ' + Mode('$os').Get());

var result = Modal('$os', {
    win32: '- Running on Windows 32',
    linux_or_linux: '- Running on Linux',
    darwin: '- Running on Darwin',
    $else: '- Running on UNKNOWN'
});

cout(result);

Mode('test').Set('hi');

var result2 = Modal('Test', {
    Hello: '- HELLO -',
    Goodbye: '- GOODBYE -',
    Hi: '- HI -',
    Bye: '- BYE -'
});

cout(result2);
